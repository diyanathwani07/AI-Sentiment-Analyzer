## ✨ Overview  

The **AI Sentiment Analyzer** is a web-based application that analyzes user input text and predicts whether the sentiment is **Positive 😊** or **Negative 😠** using Machine Learning and NLP techniques.  

It provides real-time results with confidence scores and features a modern, animated UI for a smooth user experience.  

---

## 🚀 Features  

✨ Real-time sentiment analysis  
🧠 Machine Learning powered predictions  
📊 Confidence score display  
🎨 Beautiful animated UI (Glassmorphism)  
⚡ Fast & responsive performance
